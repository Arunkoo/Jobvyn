# JobVyn Frontend

A modern **job portal frontend** built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **ShadCN UI**, integrated with backend services for **authentication**, **job listings**, **applications**, **payments**, and **utilities**.

---

## ðŸ”¹ Features

- **User Authentication:** Register, login, and protected routes.
- **Job Listings:** Browse active jobs with filters by title & location.
- **Company & Job Management:** Recruiters can create/update companies & jobs.
- **Job Applications:** Users can apply for jobs, and recruiters can manage applications.
- **Payment Integration:** Subscription handled via Razorpay.
- **Notifications & Emails:** Users get status updates on applications via email.
- **File Uploads:** Resume & company logos uploaded via Cloudinary.
- **Theme Support:** Light/dark mode with `next-themes`.

- **Frontend:** Next.js, React, TypeScript
- **Styling:** Tailwind CSS, ShadCN UI, Radix UI components

- **Dev Tools:** ESLint, TypeScript

**Backend Services:** Job Service, Payment Service, Utils Service (Node.js, Express, PostgreSQL, Kafka, Razorpay, Cloudinary)
